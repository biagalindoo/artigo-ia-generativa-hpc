\begin{abstract}
Generative artificial intelligence workloads demand computational infrastructure with high processing, memory, storage, and communication capacity. This paper presents an experimental analysis of a local hardware environment for a representative matrix multiplication workload, relating the results to concepts of computer architecture and high-performance computing. The evaluated machine contains an Intel Core i7-1360P processor, 32 GB of RAM, NVMe storage, and integrated Intel Iris Xe graphics. The benchmark was implemented with Python 3.11.9 and NumPy 2.1.2 using dense float32 matrices of orders 512, 1024, and 2048. The measured average performance increased from 92.227 GFLOPS to 306.783 GFLOPS as the matrix size grew. The results provide an educational baseline for discussing how hardware infrastructure affects computational workloads associated with generative AI.
\end{abstract}

\begin{resumo}
Cargas de inteligência artificial generativa demandam infraestrutura computacional com alta capacidade de processamento, memória, armazenamento e comunicação. Este artigo apresenta uma análise experimental de um ambiente local de hardware a partir de uma carga representativa de multiplicação de matrizes, relacionando os resultados a conceitos de arquitetura de computadores e computação de alto desempenho. A máquina avaliada possui processador Intel Core i7-1360P, 32 GB de RAM, armazenamento NVMe e GPU integrada Intel Iris Xe. O benchmark foi implementado com Python 3.11.9 e NumPy 2.1.2, usando matrizes densas em \textit{float32} de ordens 512, 1024 e 2048. O desempenho médio medido cresceu de 92,227 GFLOPS para 306,783 GFLOPS conforme o tamanho da matriz aumentou. Os resultados fornecem uma linha de base educacional para discutir como a infraestrutura de hardware afeta cargas computacionais associadas à IA generativa.
\end{resumo}
